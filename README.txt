EarnYou - Premium Static Website
Files included:
- index.html
- about.html
- contact.html
- post1.html
- post2.html
- post3.html
- style.css
- search.js

How to use:
1. Unzip the folder.
2. Open index.html in a browser to preview locally.
3. To publish, upload the folder to GitHub Pages, Netlify, or Vercel.
4. To add more posts: create a new postX.html and add { title, url } entry into search.js posts array.

Theme: Dark premium (navy + gold accent).
